\# 🎭 b\_emotes



Ein modernes Emote-System für FiveM.



\---



\## Features



\- 📁 Große Auswahl an Emotes

\- 🎬 Animations-Menü

\- 💃 Tanz-Emotes

\- 🪑 Sitz-Emotes

\- 😴 Liege-Emotes

\- ❤️ Paar-Emotes

\- ⚡ Hohe Performance

\- 🔄 Einfache Installation

\- 🎨 Benutzerfreundliches Menü



\---



\## Installation



1\. Den Ordner `b\_emotes` in den `resources`-Ordner kopieren.

2\. In der `server.cfg` eintragen:



```cfg

ensure b\_emotes

```



3\. Server neu starten.



\---



\## Voraussetzungen



\- FiveM Server

\- ESX oder QBCore

\- OneSync empfohlen



\---



\## Konfiguration



Alle Einstellungen befinden sich in der Datei:



```text

config.lua

```



Dort können Emotes und weitere Optionen angepasst werden.
